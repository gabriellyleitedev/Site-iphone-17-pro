# Site-iPhone-17-Landing-Page-em-React-Tailwind

Projeto de um site do iPhone 17 Pro desenvolvido para aprendizado real de React, TailwindCSS e boas práticas modernas de front-end.
Explorei conceitos de componentes, estilização avançada, animações e responsividade — criando uma experiência premium inspirada no site oficial da Apple.

---

## Tecnologias Utilizadas

- **React.js** — criação de componentes reutilizáveis
- **TailwindCSS** — estilização rápida e responsiva direto na tag
- **JavaScript (ES6+)** — lógica da aplicação e manipulação de props & states
- **Framer Motion** — animações fluidas e profissionais
- **React Icons** — ícones otimizados e personalizáveis
- **Vite** — ambiente moderno e super rápido para desenvolvimento
  
---

## Funcionalidades
- Menu responsivocom animações de entrada
- Hero interativo apresentando o iPhone 17
- Sessões com foco em design, cores, desempenho e câmera
- Animações suaves utilizando Framer Motion
- Componentes reutilizáveis seguindo boas práticas do React
- Layout 100% responsivo com Tailwind
- Estilização feita dentro das tags usando classes utilitárias
- Uso de key, props e variáveis no React durante a construção

---

## Aprendizado
Durante o desenvolvimento, aprendi a:
- Estilizar elementos diretamente nas tags com classes utilitárias do Tailwind
- Trabalhar com variáveis, key e componentes no React
- Criar animações profissionais usando Framer Motion
- Organizar estrutura de pastas num projeto React com Vite
- Construir layouts avançados com grid, flex e breakpoints do Tailwind
- Melhorar responsividade, fluidez visual e experiência do usuário

---

## Como Visualizar

Você pode rodar o projeto localmente no terminal usando:

- npm install
npm run dev
ou
- cd "iphone 17"
npm run dev
